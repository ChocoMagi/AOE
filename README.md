# Silver Bot

## Setup

Create a `.env` file in the project root with:

```env
DISCORD_TOKEN=your_bot_token_here
DEV_GUILD_ID=123456789012345678
```

Notes:
- `DISCORD_TOKEN` comes from the Discord Developer Portal for your bot application.
- `DEV_GUILD_ID` is the server (guild) ID where slash commands will sync.

## Run

```bash
python bot.py
```
